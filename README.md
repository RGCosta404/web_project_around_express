# EUA Afora - Servidor Express

## Descrição

Back-end do projeto **EUA Afora**, desenvolvido no Sprint 15.  
Trata-se de uma API RESTful construída com Express.js para gerir utilizadores e cartões.

---

## Funcionalidades

A API disponibiliza endpoints para:
- Listar todos os utilizadores
- Obter um utilizador específico por ID
- Listar todos os cartões

---

## Tecnologias

- Node.js
- Express.js
- ESLint (Airbnb)
- Nodemon
